# Arboretum
3d printable tree armature generator

Make some nice trees, like this:

![arboretum user interface, showing a 3d preview of a procedurally generated tree and some sliders to configure how the tree should be generated](https://github.com/rbricheno/arboretum/blob/main/Screenshot%202026-07-19%20172255.png)

Highly configurable, explore and discover to find the tree you want.

It's all javascript.

Download it locally or use it live here:

https://rbricheno.github.io/arboretum/

Click to generate new previews, mess with the settings, find a tree you like then click to bake some 3d bark on it and download an STL for printing.

Fully vibecoded with Google AI studio and ChatGPT, which has got a bit better recently.
